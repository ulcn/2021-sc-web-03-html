# HTML 정리

## div, h1~6, p, ul, ol, li : 영역(block 요소 - layout요소)
## a, img, 글씨 : 영역안에 표현되는 내용(inline 요소)
## class, title, id 등 태그의 속성을 Attribute 라고 한다.

## Emmit 예
```html
html>head>title{Hello HTML}^body>h1{BOOLDOOK}+p{내용입니다.}

<html>
<head>
	<title>Hello HTML</title>
</head>
<body>
	<h1>Minsol</h1>
	<p>내용입니다.</p>
</body>
</html>


```
